# PDCabinet 发布说明

## 🎉 v1.0.0 正式发布

发布日期：2025年11月28日

### ✨ 新功能

#### 📸 自定义拍摄参数
- 新增光圈值、ISO、曝光时间、焦距、拍摄时间输入字段
- 手动输入参数可覆盖EXIF数据
- 优化的表单界面，支持分组显示

#### 🔒 照片不可变性
- 上传后的照片不可修改或删除
- 确保数据完整性和历史记录保存
- 照片详情页仅显示信息，无编辑功能

#### 👤 上传者信息展示
- 照片详情页显示上传者用户名
- 支持管理员和普通用户上传区分
- 增强的照片归属透明度

#### 🎨 UI/UX优化
- 现代化的渐变背景设计
- 响应式布局适配各种设备
- 优雅的交互动画效果

### 🏗️ 技术架构

#### 前端技术栈
- **框架**：Vue 3 + Composition API
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **构建工具**：Vite
- **UI样式**：现代化CSS3 + 渐变设计
- **HTTP客户端**：Axios

#### 后端技术栈
- **运行环境**：Node.js 16+
- **框架**：Express.js
- **数据库**：MySQL 8.0
- **认证**：JWT
- **文件上传**：Multer
- **图片处理**：Sharp
- **EXIF解析**：Exifr

### 📦 部署方式

#### 1. 传统部署
```bash
# 安装依赖
cd server && npm install
cd ../client && npm install

# 配置环境变量
cp server/env.example server/.env
# 编辑 .env 文件

# 启动服务
npm run dev
```

#### 2. Docker部署（推荐）
```bash
# 使用Docker Compose一键部署
docker-compose up -d

# 或使用部署脚本
./deploy.sh
```

### 🔧 环境要求

- **Node.js**：16.0 或更高版本
- **MySQL**：8.0 或更高版本
- **Docker**：20.10 或更高版本（可选）
- **Docker Compose**：1.29 或更高版本（可选）

### 🛡️ 安全特性

- **密码加密**：bcrypt加密存储
- **JWT认证**：无状态认证机制
- **CORS配置**：跨域请求控制
- **输入验证**：前后端双重验证
- **文件类型检查**：仅允许图片文件上传

### 📊 性能特性

- **图片压缩**：自动生成缩略图
- **懒加载**：瀑布流分页加载
- **缓存优化**：静态资源长期缓存
- **数据库索引**：优化查询性能
- **连接池**：数据库连接复用

### 🎯 使用场景

1. **个人作品展示**：摄影师展示作品集
2. **团队作品管理**：设计团队作品归档
3. **摄影社区**：用户分享摄影作品
4. **艺术展览**：线上艺术作品展示
